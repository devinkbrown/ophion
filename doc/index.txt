Here is the overview of the documents in the doc/ directory.

Subdirectories:

features/               - Documents about features and standards
technical/              - Technical documents about ircd internals and
                          protocol information
oper-guide/             - Sphinx/RST operator guide (build with make html)

Files:

ircd.conf.example       - A minimal working ircd.conf example
reference.conf          - Full reference configuration documenting every directive
ircd.motd               - A default ircd.motd used at install time
credits-past.txt        - Credits for the predecessors to Ophion
logfiles.txt            - Description of formatting of some logfiles
modes.txt               - User mode, channel mode, and snomask reference
server-version-info.txt - Overview of the flags shown in /version

Also in the extensions/ directory you will find extension modules and their
documentation.  See extensions/README for an overview.
